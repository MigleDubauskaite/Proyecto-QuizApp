Acerca de Quiz App

Aplicación para gestionar preguntas y realizar evaluaciones sobre temas específicos.

Puedes elegir categorías, tipos de pregunta (Verdadero/Falso, respuesta única o múltiple) y el número de preguntas por partida. Los resultados se guardan por usuario.

Quiz App — Proyecto Spring Boot con Thymeleaf, MySQL, MongoDB y API REST.
